栈的图文解析

发表于 2018-06-04 |

摘要：
栈是一种线性存储结构，它有以下特点：

  • 栈中的数据是“先进后出”（FILO,First In Last Out）的方式来进出栈。
  • 向栈中添加和删除数据都是从栈顶操作的

介绍

栈通常包含了三种操作：push,peek,pop:

  • push 是向栈中添加元素
  • peek 是返回栈顶的元素
  • pop 返回并删除栈顶元素的操作
